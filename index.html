PUBLIC BETA

Club Assistant

A Chess.com-inspired operations companion designed to help club owners organize members, prospects, clubs, messages, templates, and daily workflows.

Version 1.1.0 Public Beta

DOWNLOAD

Get Club Assistant

Club Assistant is currently distributed as a free public beta. Chrome requires manually distributed extensions to be installed through Developer Mode.

Download Club Assistant ZIP file • Chrome extension • Public beta

INSTALLATION

How to Install

  1. Download and extract the ZIP file.
  2. Open chrome://extensions in Google Chrome.
  3. Turn on Developer mode.
  4. Select Load unpacked.
  5. Choose the extracted Club Assistant folder.

BETA NOTICE

Built for club owners. Improved through real use.

This release is intended for early club-owner testing. Features may be refined as feedback is received, while the extension remains focused on club management rather than gameplay or move assistance.
